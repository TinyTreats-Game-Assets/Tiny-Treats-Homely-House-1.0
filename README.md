# Tiny Treats - Homely House set

[![Homely House set](https://img.itch.zone/aW1nLzE3MzkyMDU4LnBuZw==/original/q%2FE%2BhX.png)](https://tinytreats.itch.io/homely-house)

With this small set of cute, stylized low-poly game assets, you can create a cozy home scene. This set includes a cute stylized house, a set of fences, trees, other outdoor decorations, and a base tile to place it all on! This set  is perfect for adding a home to your RPG world or as part of your game's village.

### Features

- **16+ Stylised low-poly 3D models,** perfect for games across all platforms, including mobile.
- **Textured** using a single gradient atlas texture (1024x1024) that can be downsampled up to 128x128 for further optimization.
- Models are included as **.OBJ, .FBX** and **.GLTF,** That means the files work in most 3D game engines (including **Unity, Godot, Unreal Engine, Roblox,** and more)
- **Free for personal and commercial use,** no attribution required. (CC0 Licensed)
